Resort Automation System Using BLE-Mesh Network

Category: Embedded

This project aims to provide an automated solution for resort owners by using a BLE-Mesh network. The system allows guests to control lights and fans, with the added feature of fan speed adjustment, directly from their rooms. Additionally, it offers convenience by enabling guests to request room services such as laundry or sweeping. Resort management can seamlessly control the network, adding devices and rooms via a dedicated mobile app. The system includes a sensor device to monitor real-time environmental conditions, such as temperature, humidity, and smoke levels within the room, as well as displaying weather updates for the resort's city. This comprehensive solution enhances the resort experience for both guests and management.

Home Automation System Using MQTT and Socket Protocols

Category: Electronics Development

This home automation system prototype was developed to deliver an efficient solution for automating home appliances, leveraging MQTT and socket protocols for communication. Key features included real-time control of appliances and real-time temperature monitoring, all managed through a user-friendly web application. Built on a RISC architecture-based single-board computer (SBC), the system ensured low-latency communication, enabling seamless, real-time control for users. The result was a responsive and intuitive system designed to enhance the convenience of modern home automation.

SMART CAR With RISC V SBC

Category: Robotics & AI/ML

This project features a custom-designed PCB equipped with a dedicated microcontroller, motor driver, and headers for attaching servo motors and multiple sensors. At its core, the system is powered by a RISC-V based single-board computer (SBC) running Python scripts. It includes advanced functionalities like obstacle avoidance and environmental exploration, making it an autonomous solution. Additionally, the system can be remotely controlled through an integrated app from anywhere in the world, offering both automation and user interaction for enhanced flexibility and performance.

Smart Luggage System

Category: IoT

The Smart Luggage prototype was developed as an innovative solution for modern travel needs, incorporating advanced features to enhance security and convenience. It offered real-time luggage tracking with scheduled updates, a fingerprint lock system supporting multiple users, and individual control for added safety. The luggage also featured an integrated weight scale to measure its contents and provide weight limit suggestions. Automatic lock control further streamlined the experience, while a dedicated mobile app allowed users to lock or unlock the luggage, track its location, and monitor its weight—all from the convenience of their smartphone. This prototype merged smart technology with industrial design for a future-forward travel solution.

Smart Alarm Clock with E-Ink Display

Category: IoT

The Smart Alarm Clock prototype offers a modern solution for consumers seeking a display that’s easy on the eyes, featuring an E-Ink display that mimics paper without the harsh glare of traditional screens. Key functions include time, date, and alarm settings, with the ability to play custom MP3 alarm tones. The clock integrates seamlessly into a smart home network via MQTT protocol, allowing users to control it with voice assistants. Additionally, a web-based GUI enables remote customization of settings. The alarm clock also integrates with Philips Hue products, enabling users to create scene-based modes for a more personalized wake-up experience. This prototype combines smart home convenience with eye-friendly display technology.
